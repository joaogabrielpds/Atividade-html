Resumo – Estruturando Páginas Web com HTML

1. O que é HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar páginas web. Ele trabalha junto com CSS (estilo) e JavaScript (funcionalidade).

2. Versões do HTML

  • HTML (1992) – primeira versão
  • HTML 4.0 (1997) – introduziu CSS
  • XHTML – versão mais rígida baseada em XML
  • HTML5 (2014) – versão moderna
  • Hoje: padrão “Living Standard”

3. Estrutura básica

<!DOCTYPE html>
<html>
<head>
    <title>Título</title>
</head>
<body>
    Conteúdo
</body>
</html>

4. Elementos e Atributos

Elementos são definidos por tags. Atributos adicionam informações extras.

<a href="https://site.com">Link</a>

5. Principais Elementos

Textos

  • <h1> a <h6> – títulos
  • <p> – parágrafo
  • <b> / <strong> – negrito
  • <i> / <em> – itálico
  • <br> – quebra de linha

Listas

  • <ol> – lista ordenada
  • <ul> – lista não ordenada
  • <li> – item

Links

Usa a tag <a> com atributo href para navegar entre páginas.

Imagens

<img src="imagem.jpg" alt="Descrição">

Tabelas

  • <table> – tabela
  • <tr> – linha
  • <td> – célula
  • <th> – cabeçalho

Formulários

  • <form> – formulário
  • <input> – campos
  • <textarea> – texto longo
  • <select> – lista

6. Outros Conceitos

  • Elementos block e inline
  • Comentários: <!-- comentário -->
  • Caracteres especiais: &lt; e &gt;

7. Conclusão

HTML define a estrutura das páginas web usando elementos e atributos. É a base do desenvolvimento web.
